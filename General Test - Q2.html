Conjunto de dados¶

O conjunto de dados escolhido para a realização desta tarefa foi o Heart Disease UCI.

Este é um subconjunto de 14 atributos de um banco de dados que contém 76. O campo "target" refere-se à presença de doença cardíaca no paciente. É um número inteiro com valor de 0 (sem presença) e 1 (com presença).

Descrição do conjunto de dados¶

1. age - idade em anos.
2. sex - sexo (1 = masculino; 0 = feminino).
3. cp - tipo de dor no peito
    Valor 0: angina típica;
    Valor 1: angina atípica;
    Valor 2: dor não anginosa;
    Valor 3: assintomática

Descrição do conjunto de dados¶

4. trestbps - pressão sanguínea em repouso
5. chol - colesterol sérico em ml/dl 
6. fbs - açúcar no sangue em jejum maior que 120 ml/dl
7. restecg - resultados eletrocardiográficos em repouso
8. thalach - frequência cardíaca máxima alcançada
9. exang - angina induzida por exercício (1 = sim; 0 não)

Descrição do conjunto de dados¶

10. oldpeak - depressão ST induzida por exercício em
relação ao repouso
11. slope - inclinação da extremidade do segmento ST
no exercício
    Valor 1: inclinação para cima;
    Valor 2: plano;
    Valor 3: inclinação para baixo

Descrição do conjunto de dados¶

12. ca - número de vasos coloridos pela fluoroscopia (0-3)
13. thal - Talassemias 
    Valor 3: normal;
    Valor 6: defeito permanente (irreparável);
    Valor 7: defeito reversível (reparável)
14. target - diagnóstico de doença cardíaca 
    Valor 0: estreitamento de diâmetro <50%;
    Valor 1: estreitamento de diâmetro> 50%

def pca_2d_plot(X_r, labels):
    
    plt.figure(figsize=((8,4)))
    colors = ['turquoise', 'red']
    lw = 2

    for color, i, label in zip(colors, [0, 1], labels):
        plt.scatter(X_r[y == i, 0], X_r[y == i, 1],
                    color=color, alpha=.8,
                    label = label, lw=lw)
    plt.legend(loc='best', shadow=False, scatterpoints=1)
    plt.title('PCA of Heart Disease Dataset')

    plt.show()
    
def componentes_principais(df, pca, X_r):
    
    n_pcs = pca.components_.shape[0]
    most_important = [np.abs(pca.components_[i]).argmax() for i in range(n_pcs)]
    feature_names = list(df.columns)
    most_important_names = [feature_names[most_important[i]] for i in range(n_pcs)]
    return {'CP{}'.format(i+1): most_important_names[i] for i in range(n_pcs)}

def pca_3d_plot(X_r, y, labels):
    
    Xax = X_r[:,0]
    Yax = X_r[:,1]
    Zax = X_r[:,2]
    
    colors = ['turquoise', 'red']
    marker = {0:'*',1:'o'}
    fig = plt.figure(figsize=(7,5))
    ax = fig.add_subplot(111, projection='3d')
    fig.patch.set_facecolor('white')
    for color, i, label in zip(colors, [0, 1], labels):
        ix = np.where(y==i)
        ax.scatter(Xax[ix], Yax[ix], Zax[ix], 
                   c = color, marker = marker[i],
                   s = 10, label = label)
    ax.set_xlabel("Primeira componente principal", fontsize = 8)
    ax.set_ylabel("Segunda componente principal", fontsize = 8)
    ax.set_zlabel("Terceira componente principal", fontsize = 8)
    ax.legend()
    plt.title('PCA of Heart Disease Dataset')

def make_grid(model, param_grid, scoring, X, y):
   
    gs = GridSearchCV(estimator = model,
                      scoring = scoring,
                      param_grid = param_grid,
                      verbose = True,
                      n_jobs = -1,
                      refit = False).fit(X, y)
    
    grid = pd.DataFrame(gs.cv_results_)
 
    grid.rename(columns = {"rank_test_accuracy": "accuracy_rank",
                           "mean_test_accuracy": "accuracy",
                           "rank_test_precision": "precision_rank",
                           "mean_test_precision": "precision",
                           "rank_test_recall": "recall_rank",
                           "mean_test_recall": "recall",
                           "rank_test_f1": "f1_rank",
                           "mean_test_f1": "f1"},
                inplace = True)
 
    grid.sort_values(by = ["recall_rank",
                           "accuracy_rank", 
                           "f1_rank"],
                     inplace = True)
  
    return grid[["params", "recall_rank",
                 "recall", "accuracy_rank", 
                 "accuracy", "f1_rank",
                 "f1", "precision"]]

def select(grid, param = None):
    
    if param is None:
        return grid["params"].iloc[0]
    else:
        return grid["params"].iloc[0][param]

def heatmap(confusion_matrix):
    matrix_df = pd.DataFrame(confusion_matrix)
    plt.figure(figsize=((10,7)))
    ax = plt.axes()
    sns.set(font_scale=1)
    sns.heatmap(matrix_df, annot=True, fmt="g", ax=ax, cmap="magma")
    ax.set_title('Matriz de confusão - Árvore de decisão')
    ax.set_xlabel("Predição", fontsize = 15)
    ax.set_xticklabels(list(labels))
    ax.set_ylabel("Condição verdadeira", fontsize=15)
    ax.set_yticklabels(list(labels), rotation = 0)
    plt.show()
